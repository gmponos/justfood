	<li>.....</li>
	<li>.....</li>
	<li>.....</li>
</ul>

<div class="pgholder btn-group" 
	data-contentID="sample1" 
	data-perPage="5" 
	data-startPage="1" 
	data-startRange="1" 
	data-midRange="5" 
	data-endRange="1" 
	data-buttonSize="btn-small"
	data-buttonClass="btn-duadua">
</div>
data-contentID: The ID of the container of the items you want to paginate.

data-perPage: Number of items to be displayed per page.

data-startPage: The page you want to be displayed first when the pagination is applied.

data-startRange: Number of page links always visible at the beggining of the nav. For instance, if you set data-startRange to 3, page links 1, 2 and 3 will allways be visible no matter which is the current page.

data-midRange: Number of page links to display at one time i.e. let's say you have a total of 50 pages, instead of displaying links to each and every page, we display a range of pages around the current page that we're on. Just a note, the data-midRange value should be an odd number so the display is symmetrical, but it can also be even if you like.

data-endRange: Number of page links always visible at the end of the nav.

data-buttonSize: The button size class.

data-buttonClass: The button color class.
